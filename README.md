# gbsv-MC1 – Sampling Theorem
Jupyter notebooks exploring the Sampling Theorem applied to Swiss grid load data.
## Requirements
- Python 3.x
- `numpy`
- `pandas`
- `matplotlib`
- `scipy`
Install with:
```bash
pip install numpy pandas matplotlib scipy
```
## Data
Swiss electricity generation/consumption data (2026) from Swissgrid:  
https://www.swissgrid.ch/en/home/operation/grid-data/generation.html
